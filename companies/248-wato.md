# Wato

**One-Liner:** Shared memory/tools/workflows for agents

**Description:** Wato provides a shared memory, tools, and workflow layer that enables multiple AI agents to collaborate effectively. Their platform gives agent teams a shared context store where they can exchange information, delegate tasks, and maintain collective state across sessions. Wato also provides a curated library of pre-built tools that agents can use — web browsing, code execution, file management, and API integrations — plus a workflow engine for orchestrating multi-agent processes. Instead of each agent operating in isolation, Wato enables the kind of team coordination that human teams take for granted: shared notes, handoff protocols, and persistent project memory.
