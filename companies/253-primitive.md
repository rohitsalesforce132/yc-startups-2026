# primitive

**One-Liner:** Communication for agents

**Description:** Primitive builds the communication protocol and infrastructure that enables AI agents to talk to each other, to humans, and to external systems in a standardized way. Their platform provides a message bus, conversation protocols, and handoff mechanisms that make multi-agent coordination seamless. Instead of each agent team building custom communication channels, Primitive offers a unified layer that handles message routing, conversation threading, priority escalation, and context passing between agents. The platform supports synchronous and asynchronous communication patterns, enabling real-time agent collaboration as well as delayed handoffs. Primitive aims to become the TCP/IP of agent communication — the foundational protocol every agent uses.
