## What They Do

Manicule automates developer relations (DevRel) for API-first companies using AI. DevRel involves writing documentation, creating code samples, answering developer questions on Discord/Stack Overflow, running hackathons, and building community. Manicule's AI generates documentation from code, creates sample apps, answers developer questions 24/7, and identifies developer champions in the community. For API companies that can't afford a full DevRel team, Manicule provides 80% of the value at 10% of the cost.
