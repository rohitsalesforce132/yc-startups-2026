## What They Do

Codag builds specialized log compression technology designed specifically for AI agent workloads. When AI agents operate at scale, they generate massive volumes of logs — every action, decision, API call, and error creates log entries. Traditional log compression doesn't work well because agent logs have different patterns than human-generated logs. Codag's compression algorithm understands agent log structures and achieves 10-20x compression without losing debugging information. For companies running thousands of agents, this cuts storage costs dramatically and speeds up log search. It's the infrastructure layer every agent platform will need as they scale.
