# Memory Store

**One-Liner:** Shared context for team's agents

**Description:** Memory Store provides a persistent shared memory layer for AI agents that need to maintain context across conversations, sessions, and team members. Their platform enables agents to store, retrieve, and share structured memories — customer preferences, project decisions, domain knowledge, and interaction histories — creating a collective intelligence that grows over time. Unlike individual LLM context windows that reset with each conversation, Memory Store gives agents long-term recall and team-wide knowledge sharing. The platform handles memory versioning, conflict resolution when agents disagree on facts, and intelligent retrieval that surfaces the right memories at the right time without overwhelming the agent's context window.
