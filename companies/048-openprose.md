# OpenProse

**Batch:** S26
**Industry:** B2B Engineering
**Tags:** developer-tools, ai-agents, framework
**One-liner:** A natural language agent framework

## What They Do

OpenProse provides a developer framework for building AI agents that are programmed using natural language instead of code. Developers describe what the agent should do in plain English (or any language), and OpenProse translates these descriptions into executable agent logic. This dramatically lowers the barrier to building AI agents — product managers, domain experts, and non-technical people can create functional agents. For companies, it means faster agent development and agents that business users can modify without involving engineering.
