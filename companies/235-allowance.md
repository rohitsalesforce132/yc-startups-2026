# Allowance

**One-Liner:** Scoped payments for AI agents

**Description:** Allowance provides payment infrastructure that enables AI agents to make purchases and payments within scoped, controlled limits. Their API gives developers the ability to issue time-limited, amount-capped, merchant-restricted payment credentials to AI agents — similar to how parents give children allowances with boundaries. When an AI agent needs to buy cloud resources, subscribe to a service, or pay for data access, Allowance ensures it can only spend what's authorized, at approved merchants, within defined time windows. The platform provides complete audit trails, real-time spending dashboards, and automatic cutoffs when limits are reached, solving the critical trust problem of letting autonomous agents control money.
